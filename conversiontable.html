Conversion Table

How to Use This Table

This table shows all the units of weight measurement you can convert to from pounds. The way that it works is easy. Search for what tou want to convert to make sure it exists. Then, use the ID number once you are in the right group of the file converter to convert. For example if i want to convert 120 pounds to saturns. I would search for saturn, find the id, go to the converter, navigate to the milky way tab and then type the id in. This is most useful for the misc tab in stupid stuff.
